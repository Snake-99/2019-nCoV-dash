# 2019-nCoV 疫情展示

本项目用于展示新型冠状病毒 2019-nCoV 疫情展示，使用 [dash](https://plot.ly/dash/) 制作。

## 使用

```bash
python app.py
```

然后打开 `http://localhost:9102/` 即可。
